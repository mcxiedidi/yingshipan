## 📖 项目简介
YingShiPan（影视盘）是一款跨平台的网盘资源管理应用，支持挂载主流网盘账号，全自动刮削网盘内影视/文档资源的元数据（封面、简介、分辨率等），并以可视化方式展示给用户。
- 🆓 全程免费，无广告、无付费解锁功能
- 🛠️ 开源可定制，代码可自由修改但**禁止商用**
- 🎨 简洁UI，支持资源检索、分类、筛选

## ✨ 核心特性
| 特性 | 详情 |
|------|------|
| 📁 多网盘挂载 | 一键绑定多家网盘，无需手动下载资源 |
| 🔍 全自动刮削 | 基于 TMDB 接口自动补全影视资源元数据 |
| 🎯 跨平台支持 | 基于 Flutter 开发，适配 Windows/macOS/Linux/移动端 |
| 📱 友好展示 | 分类展示刮削后的资源，支持关键词检索 |
| 🔒 安全可靠 | 本项目无后端，用户数据均只存储在客户端 |
| 🧑‍💻 开源可定制 | 基于 Java/Flutter 开发，代码可自由修改扩展 |

## 🚀 快速开始
### 环境要求
- 操作系统：Windows 10+/macOS 10.15+/Linux (Ubuntu 18.04+)
- 开发/运行环境：
  - Flutter 3.10+（前端）

### 网盘挂载流程
1. 启动应用后，进入「网盘管理」→「添加网盘」
2. 选择对应网盘，按提示完成账号授权/API密钥配置
3. 授权成功后，应用自动扫描并刮削资源（首次刮削耗时取决于网盘文件数量）
4. 刮削完成后，在「资源库」查看分类后的资源列表
